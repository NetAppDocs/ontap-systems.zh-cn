= 将失败的启动介质部件返回给NetApp - FAS70 和 FAS90
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
如果您的 FAS70 或 FAS90 存储系统中的某个组件发生故障，请将故障部件退回NetApp。请参阅 https://mysupport.netapp.com/site/info/rma["部件退回和更换"]页面以获取更多信息。
