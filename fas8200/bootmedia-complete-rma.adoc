= 将故障部件退回给 NetApp - FAS8200
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
按照套件随附的 RMA 说明将故障部件退回 NetApp 。 https://mysupport.netapp.com/site/info/rma["部件退回和更换"]有关详细信息、请参见页面。

FAS8200系统仅支持手动启动介质恢复程序。不支持自动启动介质恢复。
