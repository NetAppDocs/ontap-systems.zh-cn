= 将故障部件退回给NetApp - AFF C30 and AFF C60
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
如果AFF C30或AFF C60系统中的某个组件发生故障、请将故障部件退回给NetApp。 https://mysupport.netapp.com/site/info/rma["部件退回和更换"]有关详细信息、请参见页面。
