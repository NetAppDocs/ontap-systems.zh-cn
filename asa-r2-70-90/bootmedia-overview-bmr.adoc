= 启动介质恢复概述- ASA A70和ASA A90
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
ASA A70或ASA A90存储系统的启动介质恢复功能会使用配对节点的启动映像、并自动运行相应的启动菜单选项、以便在替代启动介质上安装启动映像。

如果遇到类似于以下所示的启动错误消息、则需要更换启动介质并从配对节点还原ONTAP映像。
