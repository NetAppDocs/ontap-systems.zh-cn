= 将故障部件退回给NetApp - ASA A70和ASA A90
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
如果您的ASA 70-90 系统中的某个组件出现故障，请将故障部件退回给NetApp。请参阅 https://mysupport.netapp.com/site/info/rma["部件退回和更换"]页面以获取更多信息。
