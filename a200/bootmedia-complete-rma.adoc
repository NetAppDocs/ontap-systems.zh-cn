= 将故障部件退回给 NetApp - AFF A200
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
更换启动介质后，按照套件随附的 RMA 说明中的描述，将故障部件退回给NetApp 。参见 https://mysupport.netapp.com/site/info/rma["部件退回和更换"] 更多信息请参见页面。AFF A200系统仅支持手动启动介质恢复程序。不支持自动启动介质恢复。
