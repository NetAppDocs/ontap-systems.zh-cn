= 将故障部件退回给NetApp - ASA A20、ASA A30和ASA A50
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
如果ASA A20、ASA A30或ASA A50存储系统中的某个组件发生故障、请将故障部件退回给NetApp。 https://mysupport.netapp.com/site/info/rma["部件退回和更换"]有关详细信息、请参见页面。
