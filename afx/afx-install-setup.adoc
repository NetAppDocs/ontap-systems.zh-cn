= 安装和设置您的 AFX 存储系统
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
前往 https://docs.netapp.com/us-en/ontap-afx/index.html["AFX 文档"^]了解如何安装、使用和管理您的 AFX 系统。

这 https://docs.netapp.com/us-en/ontap-afx/index.html["AFX 文档"^]包括以下信息：

* 安装和设置说明
* 配置系统的管理说明，例如配置 NAS 存储、克隆数据和调整本地存储大小。
* 有关管理系统的说明、包括管理客户端访问、保护数据和保护数据。
* 监控和故障排除说明、包括有关警报、集群事件和系统日志的信息。


每种 AFX 系统的具体维护程序请参见link:../afx-landing-maintain/index.html["AFX系统维护科"]。
