= 添加和更换I/O模块概述- AFF A70-90K
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
您可以将存储系统中发生故障的I/O模块更换为相同类型的I/O模块或不同类型的I/O模块。您还可以将I/O模块添加到具有空插槽的系统中。
