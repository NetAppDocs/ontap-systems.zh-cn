= 为 AFF 系统热添加 NS224 磁盘架的准备工作概述
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
通过确保有足够的支持 RoCE 的存储端口，配置它们以供存储使用，并优化新盘架的现有配置，准备将 NS224 盘架热添加到您的 AFF 系统。



== 准备什么

将 NS224 盘架热添加到 AFF 系统涉及在物理安装新盘架之前验证和配置存储连接基础架构。这些任务可确保您的 HA 对具有必要的支持 RoCE 的以太网端口，并确保您的系统配置为支持额外的存储容量。

您需要完成的具体准备任务取决于您的平台型号、当前配置以及要添加的磁盘架数量：

安装支持RoCE的PCIe卡或I/O模块:: 验证您的系统有足够的可用 RoCE 以太网端口（每个控制器上每个机架两个）。如果需要额外的端口，请在正确的控制器插槽中安装支持 RoCE 的 PCIe 卡或 I/O 模块。
配置支持RoCE的端口以供存储使用:: 如果要使用非专用的支持 RoCE 的端口连接 NS224 托架，请配置这些端口以供存储使用而不是网络使用。这是一个无中断的程序。
可重新对现有磁盘架进行配置:: 对于某些平台型号，您可能需要将现有机架重新连接到不同插槽中的两组端口，以便在安装其他支持 RoCE 的 PCIe 卡或 I/O 模块后提供针对插槽故障的弹性。
禁用自动驱动器分配:: 如果您计划手动分配驱动器所有权，请在热添加盘架之前禁用自动驱动器分配。
