= 将 NS224 磁盘架热添加到您的停产系统
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
请按照以下工作流程步骤将 NS224 磁盘架热添加到您的可用性终止系统中。

.开始之前
* 此过程仅适用于直连存储。要查看交换机连接存储的说明，请查看 link:cable-as-switch-attached.html["交换机连接的布线指南"]。
* 要热添加 NS224 磁盘架，您的 HA 对必须满足某些要求。查看 link:hot-add-eoa-requirements.html["热添加要求和最佳实践"]。


.image:https://raw.githubusercontent.com/NetAppDocs/common/main/media/number-1.png["一个"]link:hot-add-eoa-requirements.html["查看热添加要求"]
[role="quick-margin-para"]
请验证您的可用性终端系统是否符合热添加 NS224 磁盘架的要求，包括 ONTAP 版本支持、磁盘架数量和布线要求。

.image:https://raw.githubusercontent.com/NetAppDocs/common/main/media/number-2.png["两个"]link:hot-add-eoa-prepare-overview.html["准备热添加磁盘架"]
[role="quick-margin-para"]
根据您的平台型号，您可能需要安装其他支持 RoCE 的 PCIe 卡或 I/O 模块，为存储使用配置非专用支持 RoCE 的以太网端口，并在手动分配驱动器所有权时禁用自动驱动器分配。

.image:https://raw.githubusercontent.com/NetAppDocs/common/main/media/number-3.png["三个"]link:hot-add-eoa-install.html["安装磁盘架"]
[role="quick-margin-para"]
安装磁盘架的导轨套件，将磁盘架安装并固定在电信机架或机柜中，连接电源线为磁盘架供电，并分配唯一的磁盘架 ID，以确保磁盘架在 HA 对中是独一无二的。

.image:https://raw.githubusercontent.com/NetAppDocs/common/main/media/number-4.png["四个"]link:hot-add-eoa-cable-overview.html["为磁盘架布线"]
[role="quick-margin-para"]
为要热添加的磁盘架布线、使其与HA对中的每个控制器具有两个连接。

.image:https://raw.githubusercontent.com/NetAppDocs/common/main/media/number-5.png["五个"]link:hot-add-eoa-complete.html["完成热添加"]
[role="quick-margin-para"]
如果在准备热添加时禁用了自动驱动器分配、则需要手动分配驱动器所有权、然后根据需要重新启用自动驱动器分配。
