= 将故障部件退回 NetApp - AFX 2K
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
如果 AFX 2K 存储系统中的组件出现故障，请将出现故障的部件返回给 NetApp。有关更多信息，请参阅 https://mysupport.netapp.com/site/info/rma["部件退回和更换"]页面。
