= 自动启动介质恢复概述—AFF C80
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
您可以通过自动启动介质恢复(B融合)选项更换发生故障的启动介质。

自动启动介质恢复会使用配对节点中的启动映像、并自动运行相应的启动菜单选项、以便在替代启动介质上安装启动映像。
