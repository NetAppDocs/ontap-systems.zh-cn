= 冷更换 DS212C、DS224C 或 DS460C 磁盘架
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
在使用磁盘的生产系统中更换驱动器盘架时，需要执行冷盘架更换。此操作会造成系统中断，需要您停止 HA 对中的控制器。

本程序适用于带有 IOM12、IOM12B 或 IOM12C 模块的磁盘架。

使用NetApp知识库文章 https://kb.netapp.com/onprem/ontap/hardware/How_to_replace_a_shelf_chassis_using_a_cold_shelf_removal_procedure["如何使用冷磁盘架拆卸操作步骤更换磁盘架机箱"]。
